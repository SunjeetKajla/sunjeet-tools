Sunjeet Tools Documentation
===========================

Personal utility library with common development tools.

Installation
------------

.. code-block:: bash

   pip install sunjeet-tools

Quick Start
-----------

.. code-block:: python

   from sunjeet_tools import gacp
   
   # Git add, commit, and push in one line
   gacp(".", "Initial commit", "main")
